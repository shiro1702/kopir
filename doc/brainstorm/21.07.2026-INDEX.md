# Индекс: брейншторм 21.07.2026 — выбор точки печати

Сырой диалог: [21.07.2026.md](./21.07.2026.md) (не редактировать).

## Суть

Превратить блок «точки печати» из карты с маркерами в **единый механизм выбора точки** на трёх каналах с одинаковой логикой:

| Канал | Роль |
|-------|------|
| **Сайт** | Лендинг, SEO, выбор точки → deep link в бота |
| **Mini App** | Выбор точки в заказе (список + карта + bottom sheet) |
| **Бот** | Fallback: inline-список, код, last point (см. [bot-point-selection.md](../product/bot-point-selection.md)) |

Структурированная спека: [product/point-picker-ux.md](../product/point-picker-ux.md)
