# BOM прототипа бокса (на столе)

Источник: [брейншторм 24.06.2026](../brainstorm/24.06.2026.md)

Стратегия: тест на МФУ знакомых в копицентре → свой принтер после партнёрских доходов → коробка МДФ.

## Итого

| Вариант | ₽ |
|---------|---|
| Без модема (дом/Wi-Fi копицентра) | ~4 600 |
| С модемом (выезд «в поля») | ~6 600 |

Принтер на Avito (HP 1010/1020, P1102, Canon LBP2900): **3 000–5 000 ₽** — после тестов софта.

## План прототипа

1. **Закупка железа** — пока идёт доставка:
2. **VPS** (~150–250 ₽/мес): бот, LibreOffice headless, подсчёт страниц PDF
3. **Связка Orange Pi ↔ сервер** — WebSocket / polling
4. **Тест в копицентре** — МФУ по USB, CUPS
5. **Коробка МДФ** — вентиляция, доступ к бумаге сзади, лоток выдачи без доступа внутрь

## Интернет

- **Вариант А:** Wi-Fi здания (бесплатно)
- **Вариант Б (MVP):** 4G модем Huawei E3372 — независимость от Wi-Fi вуза

## Проектирование коробки (будущее)

- Вентиляция (принтер до 180°C)
- Обслуживание бумаги сзади/сбоку
- Окно выдачи без доступа рукой внутрь
