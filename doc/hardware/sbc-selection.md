# Выбор одноплатника (SBC) для бокса

Источник: [брейншторм 24.06.2026](../brainstorm/24.06.2026.md)

Задачи SBC: принять файл из бота, печать через USB, статус принтера (бумага/тонер).

## Сравнение

| Плата | Цена | Плюсы | Минусы | Вердикт |
|-------|------|-------|--------|---------|
| **Orange Pi Zero 3** (1.5–2 GB) | 2–3.5k | Дёшево, Wi-Fi, EMMC | Меньше доков | **Старт MVP** |
| **Orange Pi 3 LTS** | 2.5–3.5k | Ethernet, USB 3 | — | Бутстрэппер |
| **Raspberry Pi 4** (2–4 GB) | 6.5–8.5k | Комьюнити, стабильность | Дорого | Если бюджет / б/у |
| **Raspberry Pi 3 B+** б/у | 3–4k | Драйверы, достаточно мощи | Медленнее Pi 4 | Компромисс |
| **Raspberry Pi Zero 2 W** | 2.5–3k | Мини | 512 MB, micro-USB хабы | **Не для печати** |

## Софт-стек

- **ОС:** Linux (Armbian / Debian)
- **CUPS:** драйверы принтеров, `lp file.pdf`
- **Python:** API к Telegram-боту, очередь печати

## RAM: 1.5 GB vs 2 GB

При архитектуре «умное облако + тонкий клиент» (конвертация Word на сервере):

| Версия | Цена | Когда |
|--------|------|-------|
| 1.5 GB | ~3000 ₽ | MVP без камеры, пик ~500–600 MB |
| 2 GB | ~5500 ₽ | Запас под камеру, тяжёлые PDF |
| 4 GB | — | Избыточно для печати |

**Рекомендация MVP:** Orange Pi Zero 3 **1.5 GB** за ~3000 ₽ (экономия 2500 ₽ на комплектующие).

## Пути старта

- **Экономный:** Orange Pi Zero 3 2GB с AliExpress ~2200 ₽
- **Быстрый:** б/у Raspberry Pi 3/4 на Avito — меньше возни с драйверами
