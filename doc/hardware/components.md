# IoT-компоненты бокса

> Источник: [brainstorm/12.06.2026.md](../brainstorm/12.06.2026.md)

## Индикация и UX

| Компонент | Назначение |
|-----------|------------|
| LED WS2812B | Статусы: готов / печать / нет бумаги / ошибка / «забери документ» |
| OLED/LCD | PIN для заправщика, IP, ошибки |
| Кнопка «Помощь» | Отмена / вызов поддержки |

## Безопасность корпуса

| Компонент | Назначение |
|-----------|------------|
| Концевики в ножках | Тампер — снятие с пола |
| MPU6050 (акселерометр) | Удар / опрокидывание → сирена + алерт |
| Геркон на двери склада | Взлом / долго открыта |
| Стальной трос + якорь | Физическая защита от уноса |

## Запирание

| Режим | Описание |
|-------|----------|
| Штатный | Электромагнит 12V, импульс от Orange Pi по команде бота |
| Аварийный | Механический цилиндр + трос к защёлке; ключ только у владельца франшизы |

## Мониторинг расходников

- **Программный учёт:** счётчик страниц через API принтера + админ-бот
- **LED как индикатор остатка:** жёлтый < 20%, красный < 5%
- Опционально: ToF/ультразвук уровня бумаги (дороже, не MVP железа)

## Опционально (позже)

- ИК-датчик «забытая бумага» в лотке выдачи
- NFC RC522 — студенческие карты, скидки
- Вентилятор по датчику температуры
- Голосовой модуль для подсказок

## Дипломный сезон

- Два принтера в одном боксе (резерв) или PRO-DIPLOMA с двумя лотками
- Склад на 5000+ листов — обслуживание раз в несколько недель
