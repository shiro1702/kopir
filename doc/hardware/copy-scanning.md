# Ксерокопирование в боксе (отложено)

Источник: [брейншторм 24.06.2026](../brainstorm/24.06.2026.md)

**Статус:** отложено. Сначала простой вариант — только печать файлов из бота.

Технически реализуемо на Orange Pi через **SANE** + CUPS:

```
scanimage --format=jpeg > /tmp/scan.jpg
→ convert to PDF
→ lp /tmp/scan.pdf
```

## Проблема: антивандальность

| Риск | Описание |
|------|----------|
| Воровство бумаги | Доступ к лотку через крышку сканера |
| Вандализм | Стекло, кофе, жвачка |
| Забытые оригиналы | На стекле |

## Варианты конструкции

### 1. ADF (автоподатчик) — рекомендуемый

- Сканер внутри, снаружи только щели приёма/выдачи
- МФУ от 8–12k ₽ на Avito
- Паспорт не скопируешь (только листы А4)

### 2. Открытая ниша со стеклом

- Окно в МДФ под стекло сканера
- Паспорта, зачётки — можно
- Закалённое стекло поверх родного

## UX в боте

1. «Сделать ксерокопию»
2. «Положите документ / вставьте в приёмник → Готово»
3. Оплата (напр. 10 ₽/стр.)
4. Скан → печать

## Резюме

Фича выделяет бокс среди «только печать с флешки». Протестировать на МФУ в копицентре до проектирования корпуса.
