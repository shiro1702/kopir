## 1. `/partners` — для копицентров

**H1:** Автоматизируйте печать А4 ч/б в вашем копицентре  
**Title (SEO):** Автоматизация печати А4 ч/б для копицентров — Telegram-бот и QR-заказы

| # | Блок |
|---|------|
| 1 | Hero: оффер + CTA «Подключить точку бесплатно» + схема QR → бот → файл → оплата → печать |
| 2 | Боль: мелкая печать отнимает время оператора |
| 3 | Решение: самообслуживание А4 ч/б |
| 4 | Что автоматизируем: только А4 ч/б (формат, копии, фикс. цена, оплата до печати) |
| 5 | Что **не** автоматизируем: цвет, фото, ламинирование, переплет, плавающие цены |
| 6 | Информационный стенд точки (адрес, график, доп. услуги «от») |
| 7 | Выгоды: меньше рутины, предоплата, понятные заказы, QR-канал |
| 8 | Подключение за 1–2 дня (5 шагов) |
| 9 | Условия пилота: 0 ₽ подключение, комиссия только с заказов |
| 10 | FAQ (5–7 вопросов) |
| 11 | Финальный CTA + контакты |

**Экономика на лендинге:** 75% партнёру на старте ([partner-economics.md](./partner-economics.md)).

## 2. `/print` — для клиентов

**H1:** Распечатай документ А4 ч/б через Telegram  
**Title (SEO):** Распечатать документ А4 ч/б через Telegram — загрузите файл онлайн

| # | Блок |
|---|------|
| 1 | Hero: «за 1 минуту» · цена от · CTA «Распечатать в Telegram» + QR |
| 2 | 3 шага: файл → копии → оплата → забор |

| 5 | Цены: А4 ч/б от X ₽/стр., зависит от точки |
| 6 | Где забрать: карточки точек / ссылка на `/print/{city}` |
| 7 | Другие услуги точки — информация, не автозаказ |
| 8 | FAQ |
| 9 | Финальный CTA |

**Важно:** «онлайн» = оформление онлайн, забор на точке.

## 4. Главная `/`

**H1:** Быстрая печать документов А4 ч/б через Telegram  
**Title (SEO):** Печать документов А4 ч/б через Telegram — быстро и без флешки  
**Description:** Загрузите документ в Telegram-бот, выберите количество копий, оплатите онлайн и заберите распечатку в точке печати.

Задача: [sprint-4/10-homepage-landing.md](../../sprints/sprint-4/tasks/10-homepage-landing.md)

### Шапка

Навигация: **Как работает** · **Точки печати** · **Для копицентров** · **FAQ**  
Кнопки: «Распечатать в Telegram» · «Подключить точку»

### Блоки страницы

| # | Блок | Содержание |
|---|------|------------|
| 1 | Hero | H1 + подзаголовок · бейджи (А4 ч/б, без флешки, цена до оплаты) · двойной CTA · сноска «только А4 ч/б» |
| 2 | Что это за сервис | Простое объяснение; не заменяем весь копицентр |
| 3 | Как работает | 4 шага: бот → файл → копии → оплата → забор |
| 4 | Что умеет бот | Список возможностей **текущей версии**; честно про roadmap (диапазон страниц — позже) |
| 5 | Для каких документов | Заявление, справка, билет, договор, реферат… |
| 6 | Цена | А4 ч/б от X ₽/стр.; сумма в боте до оплаты |
| 7 | Точки печати | Карточки активных точек или «пилотный режим» |
| 8 | Другие услуги точки | Справочно: цвет, ламинирование… — у оператора |
| 9 | Для копицентров | Короткий B2B-блок → CTA + ссылка «Подробнее» на `/partners` |
| 10 | Инфо-стенд | Что даёт страница точки (адрес, график, услуги, QR) |
| 11 | FAQ | 6–8 вопросов (клиент + SEO) |
| 12 | Финальный CTA | «Распечатать в Telegram» + «Подключить копицентр» |

### Deep links бота

Один бот, разные `start`:

- Клиент: `?start=print` (или без payload)
- Партнёр: `?start=partner`

CTA «Распечатать» — в hero, после «Как работает», в карточке точки, финальный блок.

### Футер (обязательно)

- © год, **ИП / наименование**
- **ИНН**, **ОГРНИП** (из `useLegalEntity()`)
- Email, Telegram поддержки
- Ссылки: [Оферта](/offer) · Политика конфиденциальности (когда будет) · Партнёрам

> При онлайн-оплате футер с реквизитами и офертой — обязателен.
