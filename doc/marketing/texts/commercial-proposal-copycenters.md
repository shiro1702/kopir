# Коммерческое предложение для копицентров

**Кому:** владелец / управляющий копицентра  
**От:** платформа быстрой печати Kopir

---

## Проблема: на что тратят время сотрудники?

Каждый день — десятки рутинных действий:

- Принять флешку (риск вирусов)
- Назвать почту / Telegram для файла
- Открыть файл, настроить печать
- Принять копеечную оплату, выдать чек

На заказ А4 — до **5 минут** администратора. В это время теряются клиенты на баннеры, фото, сувенирку.

---

## Решение: автоматическая печать через Telegram-бот

Клиент отправляет документ в бот, выбирает параметры, оплачивает в боте — **принтер на точке печатает сам**.

Сотрудник не прикасается к компьютеру: забирает листы из лотка и отдаёт клиенту.

---

## Что получает партнёр

| Выгода | Описание |
|--------|----------|
| Новый поток клиентов | Маркетинг на нас — студенты, офисные, курьеры |
| Пассивный доход | **80%** с каждого листа (наша комиссия **20%**) |
| Экономия времени | Админ занимается сложными заказами |
| Безопасность | Никаких чужих флешек |

---

## Подключение: 3 шага

1. **Установка ПО** — лёгкая программа на Windows (фон, не грузит систему)
2. **Настройка** — выбор принтера + токен точки
3. **Готово** — принтер принимает заказы из бота

---

## Панель управления

- **Баланс** — сколько заработал копицентр сегодня / за месяц (только его доля)
- **Пауза** — один клик, бот перестаёт слать заказы
- **Печать заново** — если замяло бумагу, без повторной отправки файла клиентом

---

## Финансовый пример

| Параметр | Значение |
|----------|----------|
| Цена для клиента | 10 ₽ / стр. А4 |
| Доля партнёра | 8 ₽ (80%) |
| Себестоимость листа | ~2 ₽ |
| Чистая прибыль партнёра | ~6 ₽ / стр. |
| 100 стр./день | ~18 000 ₽ доп. дохода / мес. |

**Бесплатный тест-драйв 7 дней.** Настройка ~10 минут. Не понравится — отключаем, вы ничего не теряете.

- **Нулевой риск** — платите только с реальных заказов из бота
- Цифры адаптировать под регион
