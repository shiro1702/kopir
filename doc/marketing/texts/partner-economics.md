# Экономика партнёров

## Базовая модель (старт)

| Параметр | Значение |
|----------|----------|
| Цена для клиента | **10 ₽** / стр. А4 (партнёр может выставить свою в ЛК — позже) |
| Наша комиссия | **20%** (2 ₽) |
| Доля партнёра | **80%** (8 ₽) |
| Себестоимость листа у партнёра | ~1,5–2,5 ₽ (бумага + тонер + износ) |
| Чистая прибыль партнёра | ~6 ₽ / стр. |

## Почему не 50% на старте

Копицентр несёт все расходы: аренда, принтер, бумага, картриджи. Пока бренд неизвестен — 50% нереалистично.

20% — партнёр почти не рискует: нет заказов → ничего не платит.

## Расчёт при 10 ₽/лист

```
Клиент платит:     10 ₽
Партнёр получает:   8 ₽  (минус ~2 ₽ себестоимость = ~6 ₽ прибыль)
Мы получаем:        2 ₽  (инфраструктура, бот, платежи)
```

## Будущие модели (не на старте)

| Модель | Когда |
|--------|-------|
| Подписка 990–1490 ₽/мес | Крупные точки, >1000 листов/мес |
| Индивидуальный тариф | После админки с настройкой цены |
| Гибрид: фикс комиссия + своя цена | Масштаб |

## Индивидуальные тарифы

Обязательная фича на будущее: вуз (низкая цена, высокий поток) vs элитный ЖК (20 ₽/лист).

На старте — один фиксированный процент для простоты переговоров.
