# SEO-стратегия сайта Kopir

Источник: [брейншторм 20.06.2026](../brainstorm/20.06.2026.md)

**Ключевая идея:** Telegram-бот не индексируется → сайт перехватывает поисковый спрос и ведёт в бота.

Задачи в спринтах: [sprint-5/06](../sprints/sprint-5/tasks/06-print-point-pages.md) · [08](../sprints/sprint-5/tasks/08-client-landing-print.md) · [09](../sprints/sprint-5/tasks/09-seo-foundation.md)

---

## 1. Два направления

### B2C (клиенты)

Запросы: распечатать документ, А4 ч/б, PDF/Word с телефона, без флешки, рядом, {город}, {улица}.

**Цель:** `/print` или страница точки → CTA в Telegram.

### B2B (партнёры)

Запросы: автоматизация копицентра, Telegram-бот для копицентра, QR-заказы, приём заказов онлайн.

**Цель:** `/partners` → заявка / бот.

---

## 2. Структура URL

```
/                           # главная
/print                      # клиентский лендинг
/partners                   # B2B
/print/{city}               # точки города + карта
/print/{city}/{slug}        # страница точки (локальное SEO)
/blog/{slug}                # информационные статьи (фаза 2)
```

---

## 3. Минимальный набор для старта (6 страниц)

1. `/` — главная  
2. `/print` — распечатать документ  
3. `/partners` — для копицентров  
4. `/print/{city}` — список точек  
5. `/print/{city}/{slug}` — первая точка  
6. `/blog/kak-raspechatat-dokument-s-telefona` — первая статья  

---

## 4. Страница точки — локальное SEO

**Уникальный контент на каждой странице:** адрес, район, ориентир, график, цена этой точки, фото входа, доп. услуги, FAQ по точке.

**Schema.org:** `LocalBusiness`, `PostalAddress`, `OpeningHoursSpecification`, `Offer`, `GeoCoordinates`, `FAQPage`, `BreadcrumbList`.

---

## 5. Приоритетные запросы (не широкие)

| Тип | Примеры |
|-----|---------|
| Клиент | распечатать документ с телефона · без флешки · через Telegram · {город} · {район} |
| Партнёр | Telegram-бот для копицентра · QR-заказы · автоматизация мелкой печати |

Не гнаться за «печать документов» / «копицентр» — высокая конкуренция.

---

## 6. Технический минимум

- `robots.txt`, `sitemap.xml`
- Яндекс Вебмастер, Google Search Console
- Яндекс Метрика
- Mobile-first, CTA без прокрутки
- Человекопонятные URL, без дублей (`www`/без, trailing slash)

---

## 7. Локальное SEO

- Яндекс Бизнес / 2ГИС: карточка точки, ссылка на страницу `/print/{city}/{slug}`
- Партнёр ставит ссылку на свою страницу на сайте копицентра
- Готовый текст для партнёра — в [booklets-and-posters.md](./texts/booklets-and-posters.md)

---

## 8. Перелинковка

| Откуда | Куда |
|--------|------|
| Статья блога | `/print` |
| `/print` | `/print/{city}` (выбор точки) |
| Страница точки | `/partners` («Подключить свой копицентр») |
| `/partners` | пример страницы точки |

---
