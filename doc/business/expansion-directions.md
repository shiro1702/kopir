# Направления масштабирования платформы

Источник: [брейншторм 24.06.2026](../brainstorm/24.06.2026.md)

База: железо (микрокомпьютер, локеры), Telegram-бот с платежами, аудитория студентов в вузах.

## 1. Шеринг повербанков и зонтов

Аренда через бота: залог на карте → списание за время → разблокировка при возврате.

**Доработка:** кабель зарядки в ячейке, USB-идентификация возврата.

## 2. Локер для местных мастеров (Click & Collect)

Крафтовые мастера (бижутерия, кошельки, кроссовки) — аренда ячейки как витрина. Прозрачное стекло + подсветка.

**Модель:** помесячная аренда ячейки (~3000 ₽) или % с продажи.

## 3. Локер доставки еды (ТЦ / БЦ)

Курьеры не поднимаются на этажи; охрана не пускает. Термо-локер на ресепшене.

## 4. Реклама в вузах (B2B)

- Бесплатная печать за счёт бренда (QR на обороте)
- Рассылка в боте по вузу
- Брендирование бокса

## 5. Микро-маркет в общежитии

Шкаф 10–15 ячеек без принтера: снеки, энергетики, бытовая химия. 24/7, наценка до 200%.

## Дополнительные направления (из уточнений)

| Направление | Суть |
|-------------|------|
| Локеры ключей | Посуточная аренда, автопрокат — передача без встречи |
| Drop-off химчистка/ремонт | Сдал в ячейку → мастер забирает → возврат в локер |
| Шеринг инструмента в ЖК | Перфоратор, Karcher — аренда по RFID/весу |
| Деканат-локер | Справки об обучении без очереди в деканат |

## Общий софт (95% одинаковый)

- Авторизация Telegram
- БД ячеек (свободна/занята)
- Платежи СБП
- Команда на реле замка

Меняется корпус и UI бота.

## Логичная последовательность

1. Микро-маркет в печатном боксе
2. Рекламная монетизация в боте
3. Масштаб шкафов в общежития
4. SaaS / франшиза локеров (еда, химчистка, ключи)
