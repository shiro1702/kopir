# Бизнес-модели Kopir

> Источники: [brainstorm/12.06.2026.md](../brainstorm/12.06.2026.md), [brainstorm/13.06.2026.md](../brainstorm/13.06.2026.md)

## Две модели

| | **Модель А: SaaS** | **Модель Б: Франшиза боксов** |
|---|-------------------|------------------------------|
| Суть | Софт на ПК копицентра | Физический бокс + софт |
| CapEx | ~0 ₽ | ~30–93 тыс. ₽/бокс (см. [hardware/configurations.md](../hardware/configurations.md)) |
| Запуск | 2–3 недели | 2–3 месяца (сборка, логистика) |
| Поддержка | Низкая (только софт) | Высокая (бумага, ремонт, вандализм) |
| Потенциал | Средний чек | Высокая маржа + паушальный взнос |

## Принятое решение (июнь 2026)

**Гибридный старт:**

1. **Сейчас (Sprint 0):** SaaS на ПК дружественного копицентра — 0 ₽ железа, отладка софта.
2. **После MVP:** первый собственный бокс в вузе Улан-Удэ.
3. **Масштаб:** SaaS в новые города + продажа франшизы боксов.

## Оплата

- **Боксы:** только СБП / оплата в боте + QR на корпусе. Без POS-терминала.
- **Копицентры:** Т-Банк СБП через Mini App (Sprint 2).

## Рынок старта

**Улан-Удэ:** ~440 тыс. жителей, ~35 тыс. студентов, 40–50 копицентров.

Расширение: Иркутск, Чита — см. [economics.md](./economics.md).

## Франшиза LITE-BYOP (идея, июнь 2026)

Пitch для франчайзи:

> «Купи **3 настольных LITE-BYOP** (~28k ₽ каждый) по цене одного MAX. Поставь в 3 общежития. Обслуживание — смена картриджа раз в полгода; бумагу носят студенты».

- CapEx на 3 точки: ~84 000 ₽ vs ~72 000 ₽ один MAX
- Нулевой OPEX на бумагу; маржа = тонер + амортизация принтера
- Гибридная версия (наша + своя бумага) — для точек с заправщиком: см. [byop-hybrid-printing.md](../product/byop-hybrid-printing.md)
