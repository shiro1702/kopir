# Выплаты партнёрам: функционал сервиса

Источник: [брейншторм 30.06.2026](../brainstorm/30.06.2026.md)

**Связанные документы:** [agent-payments-tbank.md](./agent-payments-tbank.md) · [tbank-onboarding.md](./tbank-onboarding.md) · [website-structure.md](../product/website-structure.md)

Схема: **агентский договор** — все платежи на ИП агента, выплаты партнёрам по отчёту. См. также обход без номинального счёта на старте в [agent-payments-tbank.md](./agent-payments-tbank.md#старт-без-номинального-счёта-mvp).

---

## 1. БД: точка ↔ партнёр

В карточке `Point`:

| Поле | Описание |
|------|----------|
| `partner_id` | Владелец точки |
| `payout_type` | `percent` (70/30) или `per_page` (фикс ₽/стр.) |
| `partner_requisites` | ИНН, Р/С, БИК, название ИП/ООО |

---

## 2. Виртуальный баланс партнёра

При успешной оплате заказа — мгновенное начисление в регистре:

```
Клиент заплатил 100 ₽ (условия 70/30)
→ комиссия сервиса: 30 ₽
→ баланс партнёра: +70 ₽
```

Не ждать конца месяца для учёта — только для фактической выплаты.

---

## 3. Кабинет партнёра

Telegram-бот для партнёров или раздел в web/LK:

| Экран | Содержимое |
|-------|------------|
| Статус принтеров | 🟢 работает / 🔴 нет бумаги |
| Статистика | стр./день, неделя, месяц |
| Баланс к выплате | сумма на текущий момент |
| Действие | `💸 Запросить выплату` (опционально) |

---

## 4. Отчёт агента (1 клик)

Кнопка в super-admin: **«Сформировать отчёты за месяц»**.

На каждого партнёра — Excel/PDF:

> Агент за [месяц] принял платежей 50 000 ₽. Комиссия агента 15 000 ₽. К выплате принципалу 35 000 ₽. Детализация по дням — в приложении.

Скачать → подпись партнёра → перевод по реквизитам.

---

## 5. Выплаты через Т-Бизнес

### MVP (полуручной)

1. Админка: список «Партнёр 1: 12 500 ₽», «Партнёр 2: 8 300 ₽»
2. Экспорт реестра (.txt / .xml для 1С)
3. Загрузка в Т-Бизнес → массовые платежи → подпись СМС

### PRO (позже)

Open API Т-Банка — автоматические выплаты по расписанию.
