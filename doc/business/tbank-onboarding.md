## ОКВЭД (добавить к ИП)

| Код | Назначение |
|-----|------------|
| **82.19** | Копирование, подготовка документов (основной для печати) |
| **18.12** | Полиграфическая деятельность |
| **47.99** | Розничная торговля вне магазинов (вендинг / автоматы) |

### Как изменить через Т-Бизнес

1. ЛК Т-Бизнес → **Настройки** → **Регистрационные данные** → **Изменить коды ОКВЭД**
2. Добавить коды, подписать форму Р24001 по СМС (КЭП банка)
3. ФНС — до 5 рабочих дней, придёт обновлённый лист ЕГРИП

## Чеклист перед приёмом денег в боте

### 1. Т-Касса (интернет-эквайринг)

- Раздел «Т-Касса» / «Интернет-эквайринг» в Т-Бизнес
- Способ интеграции: **Telegram-бот**
- Получить `TerminalKey` + пароль → `web/.env`
- См. Sprint 3: Init / GetQr / webhook

### 2. Облачная касса (54-ФЗ)

Физический ККТ не нужен — аренда облачной кассы:

- Партнёры Т-Банка: АТОЛ Онлайн, CloudKassir, Orange Data
- ~1500–2000 ₽/мес + фискальный накопитель (ФН 15/36 мес)
- Цепочка: Т-Касса → облачная касса → чек клиенту + ФНС

### 3. СБП

В Т-Кассе **обязательно** включить СБП:

- Комиссия карт ~2.5–3%, СБП ~0.4–0.7%
- UX: кнопка в боте → приложение банка → оплата в один клик
