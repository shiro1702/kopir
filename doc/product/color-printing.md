# Цветная печать

> Источник: [brainstorm/12.06.2026.md](../brainstorm/12.06.2026.md)  
> **Статус:** отложено. Старт — только ЧБ (копицентры и боксы).

## Почему не сейчас

- Высокая стоимость ошибки (цветной тонер, недовольный клиент)
- Струйники в боксах засыхают при простое
- Сложный постраничный анализ PDF

## Целевая архитектура (этап 3+)

### 1. Постраничный анализ на сервере

- Скан RGB/CMYK каждой страницы
- Страница с цветом вне grayscale → «цветная»
- В Mini App: разбивка цены + кнопка «Сделать всё ЧБ»

### 2. Боксы

- Только **лазерные цветные** (Canon / HP entry-level)
- Мониторинг 4 тонеров (CMYK) через IPP/SNMP → алерты в админ-бот

### 3. Копицентры (SaaS)

**Умная маршрутизация:**

- Один МФУ → grayscale/color per page в драйвере
- Два принтера → split PDF: ЧБ на быстрый HP, цвет на Epson

### 4. Тарифы в БД

Отдельные цены: ЧБ / цвет × односторонняя / дуплекс на точку.
