# Индикация ячеек локера

Источник: [брейншторм 24.06.2026](../brainstorm/24.06.2026.md)

Порты на плате не заканчиваются: **I2C-мультиплексор** (TCA9548A) — 2 сигнальных порта на 8–16 экранов.

## Вариант А: OLED 0.96"

| | |
|---|---|
| Цена | 150–200 ₽/шт опт |
| Плюсы | Дёшево, светится, иконки, цена |
| Минусы | На солнце слепнет |
| Бюджет 16 ячеек | ~3000 ₽ |

## Вариант Б: E-Ink

| | |
|---|---|
| Цена | 500–800 ₽/шт |
| Плюсы | Премиум, «книжно», не жрёт при статике |
| Минусы | Медленное обновление (~1 с) |
| Бюджет 16 ячеек | ~10–12k ₽ |

## Альтернатива 1: WS2812B (адресная лента)

- 1 порт на весь шкаф
- Зелёный = свободна, красный = занята, синий мигает = твоя бронь
- ~300 ₽ на шкаф, вау-эффект как у Amazon Locker

## Альтернатива 2: Один экран 7–10" на модуль

Карта ячеек как в кинотеатре + QR. Меньше проводов, надёжнее.

## Вердикт

| Приоритет | Решение |
|-----------|---------|
| Бюджет + надёжность | LED + вся инфо в боте на телефоне |
| Вау-эффект | OLED на каждую ячейку |
| Премиум | E-Ink или один большой экран |

Телефон клиента — лучший бесплатный экран.
