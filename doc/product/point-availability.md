# Доступность точки: статусы и автопереключение

Источник: [брейншторм 30.06.2026](../brainstorm/30.06.2026.md)

**Связанные документы:** [bot-point-selection.md](./bot-point-selection.md) · [FEATURES.md](../project/FEATURES.md) (MON-01, MON-02) · [sprint-1/tasks/04-heartbeat-offline.md](../sprints/sprint-1/tasks/04-heartbeat-offline.md)

---

## Зачем

Принтеры ломаются, заканчивается бумага, пропадает связь. **Нельзя** принимать оплату на нерабочую точку — негатив, возвраты, поддержка.

Перед кнопкой «Оплатить» бот **всегда** проверяет статус выбранной точки.

---

## Статусы точки

| Статус | Значение | Источник |
|--------|----------|----------|
| `active` | Работает, можно печатать | Агент online + нет ошибок |
| `no_paper` | Нет бумаги / тонера | Агент или staff |
| `offline` | Нет связи с агентом | `lastSeenAt` / heartbeat |
| `error` | Замятие, поломка | Агент `agentStatus: error` |

Хранение: `Point.agentStatus`, `Point.lastSeenAt` (см. Sprint 1, MON-01/02).

---

## Сценарий A: последняя точка недоступна (файлы из дома)

Студент кидает файл. `last_point_id` указывает на offline/no_paper точку.

Бот **не** предлагает оплату туда. Предупреждает и предлагает альтернативу:

```
📥 Файл принят: diplom.pdf (15 стр.)

⚠️ На вашей обычной точке (ВСГУТУ, Корпус 2) сейчас нет бумаги или она временно отключена.

💡 Мы временно переключили вас на ближайшую рабочую точку:
📍 ВСГУТУ, Корпус 3 (код 103)

💰 Стоимость: 75 ₽

[💳 Оплатить и отправить на Корпус 3 (75 ₽)]
[📍 Выбрать другую точку вручную]
[❌ Удалить файл]
```

---

## Сценарий B: QR сломанной точки

Студент у принтера, сканирует QR, точка offline.

**Не** принимать файлы на эту точку. Сразу:

```
📍 Вы выбрали точку: ВСГУТУ, Корпус 2 (код 102)

❌ К сожалению, этот принтер сейчас временно не работает.

🗺 Ближайший рабочий принтер:
👉 ВСГУТУ, Корпус 3 (код 103) — работает.

[📍 Перейти на Корпус 3]
[📋 Показать другие рабочие точки]
```

---

## Логика выбора альтернативы

1. Искать `active` точки в том же ВУЗе / здании (или по координатам — ближайшие)
2. Если в здании нет рабочих — сообщение + открыть список рабочих точек **города**
3. Оплата на неактивную точку **заблокирована** на уровне API и бота
