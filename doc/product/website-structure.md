## Раздел «Печать» `/print`

| URL | Назначение | Спринт |
|-----|------------|--------|
| `/print/` | Автоопределение города → redirect | Sprint 5 |
| `/print/ulan-ude/` | Карта + список точек города | Sprint 5 |
| `/print/ulan-ude/[slug]/` | SEO-страница точки (CTA → бот, потом web-wizard) | Sprint 5 |
| `/print?point={slug}` | Wizard печати без логина | Sprint 6 |

**Страница точки (Sprint 5):**
- Адрес, фото входа, часы работы
- Цены (Ч/Б, цвет)
- Статус в реальном времени (в сети / пауза)
- Кнопка «Печатать здесь» → deep link в бот с `point_id` (Sprint 5); `/print?point=slug` (Sprint 6)
- Блок доп. услуг — информационный стенд ([point-pricing-and-services.md](./point-pricing-and-services.md))

**Карта:** фильтры «Работает сейчас», «Цветная печать».

## ЛК партнёра (Web + Mini App)

**Auth:** Telegram Login / auto в Mini App.

**Табы (нижняя навигация):**

### 1. Дашборд

- Заработано сегодня / страниц
- Мини-график за неделю/месяц
- Лента событий real-time

### 2. Мои точки

На каждую точку:
- Название, статус **В сети / Пауза**
- Токен для Windows-агента (копировать)
- Настройки: адрес, часы, **город**, координаты на карте
- Тарифы А4 (Ч/Б, цвет)
- Текстовое поле «Другие услуги и цены»

### 3. Финансы

- Баланс к выводу
- Запрос выплаты
- История выплат

### 4. Поддержка и софт

- Скачать агент Windows
- Инструкция (видео / PDF)
- Написать в поддержку (Telegram)
