# Мерчандайзинг локера (16 ячеек)

Источник: [брейншторм 24.06.2026](../brainstorm/24.06.2026.md)

Модуль расширения ~40 см ширины, крепится к печатному автомату. **0 ₽ доп. аренды** — площадь уже оплачена под принтер.

## ВУЗ: баланс «Учеба и выживание»

**50% книги (8) / 50% канцелярия и девайсы (8)**

### Книги (8 ячеек)

- 4: учебники / методички (Revenue Share)
- 4: нон-фикшн («Пиши, сокращай», «Атомные привычки»)

## Кафе / коворкинг: «Интеллектуальный отдых»

**75% книги (12) / 25% tech (4)**

| Ячейка | Товар |
|--------|-------|
| 1 | Аренда зарядки Type-C / MagSafe — 50 ₽/час |
| 2 | Крафтовый блокнот — 500–800 ₽ |
| 3 | Настольная игра (аренда) — 100 ₽/вечер |
| 4 | Зонт-шеринг — 100 ₽/сутки |
| 5–12 | Худлит, детективы, арт-книги, локальные зины |

## Правила мерчандайзинга

1. **Один слот — один фокус** (книга как на витрине + подсветка)
2. **Гравировка на стекле:** «Зарядка», «Почитай меня», «Экзамен»
3. **Динамические цены в боте:** не берут месяц → скидка автоматически

## Сводки для команды

### Локер мастеров

Витрина крафта (свечи, мерч, сладости). Revenue Share, нулевая закупка. Локации: кофейни, коворкинги.

### Микро-библиотека

Книги + канцелярия + зарядки через один бот. Настолки — не рекомендуются (потеря комплектности).
