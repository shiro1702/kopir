## Проблема

Точки сейчас создаются через seed или Prisma Studio. Для 2–3 партнёров нужно:
- завести точку без правки БД вручную;
- настроить реквизиты СБП и способы оплаты;
- сгенерировать bind-токен для staff;
- видеть online/offline агента.

Полный partner dashboard (статистика, баланс, self-service) — Sprint 3.

## Цель

Страница `/admin/points` (или вкладка в существующей админке) для владельца платформы:

| Действие | Поля |
|----------|------|
| Список точек | slug, название, online/offline, активна/нет |
| Создать / редактировать | `name`, `slug`, `pricePerPageKopeks`, `transferPhone`, `transferBankLabel` |
| Способы оплаты | `SBP_TRANSFER`, `ON_SITE`, `TBANK_ONLINE` (чекбоксы, как `PAYMENT_METHODS_ENABLED` на точку) |
| Bind-токен | кнопка «Сгенерировать» → одноразовый токен для `/bind` (задача 11) |
| QR | ссылка на deep link `?start=point_<slug>` (после Sprint 1 задача 07) |
