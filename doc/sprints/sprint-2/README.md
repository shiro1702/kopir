## Зачем задача 11 в Sprint 2

Sprint 1 использует `STAFF_TELEGRAM_CHAT_ID` / `STAFF_MAX_USER_ID` в env — достаточно для одной точки.

В Sprint 2 подключаются **2-й и 3-й партнёр**. Нужна привязка уведомлений сотрудников к **конкретной точке** без redeploy: `/bind <token>` в боте (личка или групповой чат TG).
