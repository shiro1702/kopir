## Почему не Sprint 3 (онбординг)

Пока нет онлайн-оплаты, полный partner dashboard и self-registration дают мало отдачи.  
Сначала масштабируем точки через **админку владельца** + ручная оплата, параллельно готовим Т-Банк.

## Приоритеты

```
P0: админка точек → /bind staff → токен агента → скелет Т-Банка
P1: оферта → 2–3 точки → выплаты → UX-10/MON-02 (хвост Sprint 1)
```
