## Definition of Done

- [ ] Клиент оплачивает пачку через СБП (Т-Банк) без участия staff
- [ ] Webhook переводит batch в `PAID` → агент печатает автоматически
- [ ] 2–3 точки заведены через админку (не seed/Prisma вручную)
- [ ] Staff каждой точки привязан через `/bind` (или env для пилота)
- [ ] 100+ платных транзакций за спринт

## Метрики спринта

| Метрика | Цель |
|---------|------|

| Активных точек | 3 |
