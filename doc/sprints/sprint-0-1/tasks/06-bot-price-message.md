## Цель

После calculation клиент видит страницы и сумму; при ошибке — понятный текст.

## Подзадачи

- [ ] **6.1** `messages.formatQuote(fileName, pageCount, amountRub)` 
- [ ] **6.2** `notifyQuoteReady(user, order)` — вызов из calculation handler
- [ ] **6.3** `MSG_CALCULATION_FAILED` — «Не удалось обработать файл…»
- [ ] **6.4** Формат суммы: `amountKopeks / 100` ₽, округление

## Пример сообщения

```
📄 diploma.docx
Страниц: 15
Стоимость: 150 ₽

Ожидаем оплату. После перевода сотрудник подтвердит заказ.
```

## Критерии приёмки

- [ ] Клиент получает quote в TG/MAX в течение 5 сек после calculation
- [ ] При FAILED — сообщение без суммы
