## Цель

Дать API для просмотра итогов пачки и подтверждения оплаты пачки одной операцией.

## Эндпоинты

### `GET /api/admin/batches/:id`

Возвращает:
- список файлов (`orders`)
- `totalPages`
- `totalAmountKopeks`
- статус пачки

### `POST /api/admin/batches/:id/confirm-payment`

Действие:
- проверка, что пачка в `AWAITING_PAYMENT`
- обновление `OrderBatch.paidAt`
- обновление всех `Order` пачки: `status -> PAID`, `paidAt`
