## Цель

Сделать в боте сценарий, где пользователь отправляет несколько файлов подряд и завершает набор пачки явной командой.

## Подзадачи

- [ ] **2.1** Добавить состояние чата `collecting_batch`
- [ ] **2.2** Команды:
  - «Добавить файл» (по умолчанию при входящем документе)
  - «✅ Завершить и оплатить»
  - «❌ Отменить пачку»
- [ ] **2.3** На каждый файл:
  - создать `Order`, привязать к текущему `OrderBatch`
  - назначить `batchIndex`
- [ ] **2.4** Ограничение `BATCH_MAX_FILES`
- [ ] **2.5** Таймаут неактивной пачки `BATCH_BUILD_TIMEOUT_MIN` (автоотмена)
- [ ] **2.6** Единый текст-резюме по пачке: количество файлов, страницы, сумма

## Критерии приёмки

- [ ] Пользователь может отправить минимум 2 файла до оплаты
- [ ] После «Завершить и оплатить» новые файлы не добавляются в ту же пачку
- [ ] После «Отменить» все заказы пачки переводятся в отменённый/неактивный статус
- [ ] Лимит файлов и таймаут корректно отображаются в сообщениях

## Заметки

- Сначала реализуем на Telegram, затем зеркалим в MAX
- UI сообщений сохраняем максимально простым, без inline-меню с большим числом кнопок
