## Scope

### Super-admin

- Страница «Выплаты»: список партнёров с `balanceKopeks > 0`
- Кнопка «Сформировать реестр за период»
- Экспорт `.txt` / `.xml` для загрузки в Т-Бизнес (массовые платежи)
- Отметка «Выплачено» → списание с баланса партнёра

### Partner requisites

- Поля в `Partner`: ИНН, Р/С, БИК, название ИП
- Редактирование в partner-боте или `/admin`

### Отчёт агента (ONB-07)

- PDF/Excel на партнёра за месяц
- Текст по шаблону из [partner-payouts.md](../../business/partner-payouts.md)

### Операционный процесс

1. Раз в неделю: admin формирует реестр
2. Загрузка в Т-Бизнес → подпись СМС
3. Отметка в системе → баланс обнулён

## Не в scope (Sprint 6)

- Mass Payments Open API (PAY-08)
- Кнопка «Вывести» с автопереводом

## DoD
