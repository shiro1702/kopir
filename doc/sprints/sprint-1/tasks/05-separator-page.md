# 05 — Разделительный лист (toggle)

| | |
|---|---|
| **Статус** | ⬜ todo |
| **Feature** | AGT-10 |
| **Зависимости** | 03 |
| **Оценка** | 3–4 часа |

## Цель

После каждого заказа печатается разделитель с номером заказа — сотрудник копицентра не путает выдачу.

## Подзадачи

- [ ] **5.1** Env `USE_SEPARATOR_PAGE=true|false` (default `true` для копицентра)
- [ ] **5.2** Для PDF: `pdf-lib` (Python: `pypdf` или `reportlab`) — добавить 1 страницу:
  - текст: `--- Заказ #XXXXXX ---` (shortId)
  - мелкий шрифт, по центру
- [ ] **5.3** Для DOCX: после Word print — отдельный mini-PDF separator через тот же механизм
- [ ] **5.4** Печать separator **после** основного документа, в том же job
- [ ] **5.5** Документировать: при установке в закрытый бокс — `USE_SEPARATOR_PAGE=false`

## Критерии приёмки

- [ ] 3 заказа подряд — между ними видны разделители с разными shortId
- [ ] `USE_SEPARATOR_PAGE=false` — только документ, без лишней страницы
- [ ] Разделитель попадает в лоток выдачи (ручной тест в задаче 10)

## Заметки

- Не логировать содержимое PDF
- Размер separator минимальный — быстрая печать
