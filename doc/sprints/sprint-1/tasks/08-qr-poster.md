## Цель

Партнёр печатает A4-плакат «Отсканируй → отправь файл → оплати → забери» с QR на бота и названием точки.

## Подзадачи

- [ ] **8.1** API `GET /api/points/[slug]/poster.pdf` (или `/admin/points/[slug]/poster` с `ADMIN_SECRET`):
  - A4 portrait, 210×297 mm
  - Заголовок: «Печать документов — Kopir»
  - Подзаголовок: имя точки из `Point.name`, адрес (если есть)
  - QR-код: Telegram deep link (задача 07)
  - Шаги 1–4: «Сканируй QR → Отправь PDF/Word → Оплати → Забери у принтера»
  - Footer: `@BotUsername`, цена «от X ₽/стр» из env `PRICE_PER_PAGE_KOPEKS`
- [ ] **8.2** Генерация QR: `qrcode` npm → PNG → embed в PDF (`pdf-lib` или `@react-pdf/renderer` на сервере)
- [ ] **8.3** Опционально второй QR для MAX (если используется на точке) — меньший, в углу
- [ ] **8.4** Кнопка в админке «Скачать плакат» рядом с точкой
- [ ] **8.5** Распечатать на обычном принтере партнёра, поставить на стойку (фото в [notes.md](../notes.md))
- [ ] **8.6** Проверка: телефон сканирует QR → открывается правильный бот с правильным `start`

## Критерии приёмки

- [ ] PDF открывается, QR читается с расстояния 30–50 см
- [ ] После скана — приветствие с именем точки (задача 07)
- [ ] Плакат генерируется для любого активного `Point.slug`
- [ ] Без `ADMIN_SECRET` / auth — endpoint недоступен публично (или только для admin)

## Заметки

- Не хранить PDF в Blob — генерировать on-the-fly (лёгкий шаблон)
- Брендинг минимальный: чёрно-белый, хорошо печатается на офисном принтере
- Позже: «Печать QR-материалов» из desktop GUI (AGT-11 полностью) — Sprint 2; в Sprint 1 достаточно web download
