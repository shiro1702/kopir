## Цель

Устная договорённость с 1 копицентром возле вуза на бета-тест в Sprint 1 (19 июня+).

## Критерии выбора точки

- [ ] Рядом с БГУ (Смолина/Ранжурова) или ВСГУТУ (Ключевская)
- [ ] Есть лазерный Ч/Б принтер на Windows-ПК
- [ ] Владелец/админ открыт к экспериментам
- [ ] Поток студентов в часы пик

## Подзадачи

- [ ] **9.1** Составить список 5–7 копицентров (2GIS / Яндекс.Карты)
- [ ] **9.2** Подготовить 30-секундный питч (устно)
- [ ] **9.3** Обойти 3 точки лично или написать в WhatsApp/Telegram
- [ ] **9.4** Предложение: бесплатный тест 2 недели, 80% выручки им, установка 15 мин
- [ ] **9.5** Зафиксировать: адрес, контакт, модель принтера, ОС ПК
- [ ] **9.6** Договориться о дате установки — не раньше успешного E2E (задача 08)
- [ ] **9.7** Записать в `partner-leads.md`

## Шаблон первого сообщения

```
Здравствуйте! Мы делаем сервис «печать с телефона без очереди» для студентов.
Студент сканирует QR, отправляет PDF в Telegram, оплачивает — принтер печатает сам.
Ищем 1 точку для бесплатного теста на 2 недели. Вам — 80% с каждого заказа, 
установка 15 минут на ваш ПК. Можно показать за 10 минут?
```

## Шаблон при визите

1. **Крючок:** «Сколько времени тратят сотрудники на печать 1–2 листов с флешек?»
2. **Решение:** автоматизация через Telegram, без флешек и вирусов
3. **Риск-реверс:** бесплатно, снимем за 10 минут если не понравится
4. **CTA:** «Можем поставить тест на следующей неделе?»

## Критерии приёмки

- [ ] Минимум 1 точка согласилась на тест
- [ ] Есть контакт владельца и желаемая дата старта
- [ ] Известна ОС и принтер (для подготовки Sumatra)

## Если отказывают

- Спросить: «Что должно быть, чтобы вы попробовали?»
- Предложить тест у себя дома/офисе с их принтером по записи
- Следующий кандидат из списка 9.1
