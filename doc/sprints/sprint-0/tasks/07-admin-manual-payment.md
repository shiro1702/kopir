## Цель

Простая защищённая страница: список заказов `AWAITING_PAYMENT`, кнопка «Подтвердить оплату» → `PAID` → агент подхватывает.

## Подзадачи

- [ ] **7.3** Middleware: проверка `Authorization: Bearer <ADMIN_SECRET>` или cookie session
- [ ] **7.4** Страница `app/pages/admin/index.vue`:

## Flow тестирования

1. Пользователь шлёт PDF боту → `AWAITING_PAYMENT`
2. Админ открывает `/admin`, вводит секрет (или один раз в .env на dev)
3. Жмёт «Оплатить»
4. Агент на следующем poll забирает заказ

## Критерии приёмки

- [ ] Без секрета — 401 на API и редирект/блок на странице
- [ ] Двойной клик «Оплатить» не ломает (идемпотентно или disabled)
- [ ] После оплаты заказ исчезает из списка ожидающих
- [ ] Пользователь получает сообщение в Telegram (если реализовано 7.5)
