## Как пользоваться

- Каждый спринт = **2 недели** (кроме Sprint 0 = 1 неделя)
- Задачи ссылаются на ID из [FEATURES.md](../project/FEATURES.md)
- Статусы: `⬜` / `🔵` / `✅` / `❌ cancelled`
- В конце спринта — ретро: что сработало, метрики, блокеры
