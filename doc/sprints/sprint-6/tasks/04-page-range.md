## Проблема

Клиент платит за **все страницы** файла. Частый кейс — напечатать только 1–3 страницы из большого PDF.

## Цель

Выбор непрерывного диапазона (`с` / `по`) до оплаты. Цена: `selectedPages × copies × pricePerPage`.

## Scope (план)

- [ ] **4.1** Prisma: `pageFrom`, `pageTo` (nullable = весь файл) или `selectedPageCount`
- [ ] **4.2** PDF: нарезка через `pdf-lib` на сервере перед печатью (надёжнее, чем только драйвер)
- [ ] **4.3** DOCX: конвертация в PDF + диапазон, или «только весь файл» в MVP
- [ ] **4.4** Mini App: UI с превью (FILE-04) + ползунки/инпуты
- [ ] **4.5** Бот: минимальный fallback (текст «1-5») или только Mini App

## Не в scope

- Произвольный набор страниц `1,3,7` — backlog
- Дуплекс / ч/б — UX-02, UX-03

## DoD

- [ ] PDF 20 стр., выбраны 2–4 → оплата за 3 стр. × копии → агент печатает только эти страницы
