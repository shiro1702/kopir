## Цель

Чтобы карта и карточки точки показывали актуальный контент без правок в коде: фото входа, доп. услуги, контакты, скрытие точки.

## Scope

### Admin `/admin/points/[id]`

- [ ] **6.1** Upload фото входа → Vercel Blob → `entryPhotoUrl`
- [ ] **6.2** Текстовое поле «Ориентир» (подпись к фото, напр. «вход со стороны парковки»)
- [ ] **6.3** Редактор доп. услуг: name + `priceFromKopeks` (справочно, не в калькулятор)
- [ ] **6.4** Телефон точки (`phone`) — опционально, кнопка «Позвонить»
- [ ] **6.5** Toggle «Временно скрыть с карты» (`isPublic` или `isHidden`)
- [ ] **6.6** Toggle «Принимает онлайн-заказы» (если не сделано в S5 / 11)
- [ ] **6.7** Preview карточки точки как на сайте

### API

- [ ] **6.8** `GET /api/points` — `entryPhotoUrl`, `services`, `phone`, `landmarkText`
- [ ] **6.9** Скрытые точки не отдаются публичным API

## DoD

- [ ] Партнёр/super-admin загружает фото — оно видно на `/print/.../slug`
- [ ] Услуги отображаются в модалке «Другие услуги»
- [ ] Скрытая точка не на карте и не в списке
