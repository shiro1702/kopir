# Документация Kopir

## Быстрый старт для Cursor

1. Прочитай **[PROJECT.md](./PROJECT.md)** — что строим, стек, архитектура (Neon + Vercel Blob + Vercel)
2. Настрой env: **[ENV_SETUP.md](./ENV_SETUP.md)** — где взять все ключи и токены
3. Открой **[SPRINTS.md](./SPRINTS.md)** — что делать сейчас (Sprint 0)
4. Сверяйся с **[FEATURES.md](./FEATURES.md)** — статусы фич, не дублируй работу

## Как обновлять статусы

В `FEATURES.md` меняй `⬜ todo` → `🔵 in_progress` → `✅ done`.

В `SPRINTS.md` отмечай задачи спринта и заполняй ретро в конце.

`.cursorrules` — автоматически подхватывается Cursor при работе в проекте.
